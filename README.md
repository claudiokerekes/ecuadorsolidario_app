Ecuadorsolidario App
================

Ruby on Rails
-------------

This application requires:

- Ruby 2.2.3
- Rails 4.2.6
- Posgresql

Getting Started
---------------

Configura el SMTP en secrets.yml

bundle install
rake db:create

Iniciar servidor:
-------------------------

passenger start

http://0.0.0.0:300
